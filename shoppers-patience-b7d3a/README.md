## Mode
Shortest mode

## Goal
You have an initial amount of credits and earn more each day.
Your goal is to determine how many days you must wait until you can afford all items.

## Input
Line 1: Three space-separated integers N C D

N \= number of items

C \= initial credits

D \= credits earned per day

Line 2: N space-separated integers P\_i representing the price of each item

## Output
Line 1: The minimum number of days required to have enough credits to buy all items

## Constraint
1 ≤ N ≤ 10
0 ≤ C ≤ 100
1 ≤ D ≤ 100
1 ≤ P\_i ≤ 100
