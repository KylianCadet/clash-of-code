## Mode
Fastest mode

## Goal
Write a program that takes a single string containing alphanumeric characters (letters and digits) and calculates the sum of all the numerical digits present in the string.

## Input
A single line containing a string S composed of lowercase letters (a-z) and digits (0-9).

## Output
single integer representing the sum of all digits found in the input string S.

## Constraint
The length of the string S is between 1 and 100 characters.
