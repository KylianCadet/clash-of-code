## Mode
Fastest mode

## Goal
A lightning rod has been placed in a city at the position P. This lightning rod will absorb a thunderbolt that strikes a nearby area with a distance of up to D.

More specifically, the lightning rod will absorb thunderbolts that strike a position between P - D and P + D, both ends inclusive.

Now, a thunderstorm will occur, and N thunderbolts will strike. Output the number of thunderbolts that the lightning rod will absorb.

## Input
Line 1: Three space separated integers N, P, and D.
Next N lines: An integer x that specifies where a thunderbolt will strike.

## Output
A single line containing how many thunderbolts will be absorbed by the lightning rod.

## Constraint
1 ≤ N,D ≤ 100
-100 ≤ P,x ≤ 100
